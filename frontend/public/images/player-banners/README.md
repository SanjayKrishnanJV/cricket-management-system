# Player Banner Images

This folder contains background images for player cards based on their roles.

## Required Images

Please add the following cricket images to this folder:

1. **batsman.jpg** - Cricket batting action image (e.g., batsman playing a shot)
2. **bowler.jpg** - Cricket bowling action image (e.g., bowler delivering the ball)
3. **all-rounder.jpg** - General cricket action image (versatile player)
4. **wicketkeeper.jpg** - Wicketkeeping image (e.g., keeper behind stumps)
5. **default.jpg** - Fallback cricket image

## Image Specifications

- **Format**: JPG or PNG
- **Recommended size**: 800x400 pixels
- **Aspect ratio**: 2:1 (landscape)
- **File size**: Keep under 500KB for fast loading

## Where to Find Images

You can download free cricket images from:
- **Pexels**: https://www.pexels.com/search/cricket/
- **Unsplash**: https://unsplash.com/s/photos/cricket
- **Pixabay**: https://pixabay.com/images/search/cricket/

## Current Status

Once you add the images, they will automatically appear as backgrounds on the player cards with a colored overlay matching each role:
- Batsman: Blue overlay
- Bowler: Red overlay
- All-rounder: Purple overlay
- Wicketkeeper: Green overlay
